# Used Technologies

1.  Angular v20
2.  Tailwind v4
3.  PrimeNg v20
4.  Strapi

# Prerequisites

**Latest Angular Cli installed**

```bash
npm i -g @angular/cli@latest
```

**NodeJs v22**
https://nodejs.org/en/blog/release/v22.11.0

# Instructions

1.  Inside Strapi folder `npm i`
2.  Then `npm run develop`
3.  Inside Angular app `npm i --legacy-peer-deps`
4.  Then `ng s`
